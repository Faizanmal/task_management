# Task Management Frontend Web

A modern React web application built with Next.js for task management, featuring a beautiful UI and seamless user experience.

## Features

- **Authentication**: User registration, login, and session management
- **Task Management**: Create, read, update, delete tasks with filtering and searching
- **Responsive Design**: Mobile-first design that works on all devices
- **Real-time Updates**: Instant UI updates and feedback
- **Modern UI**: Gradient design with smooth animations
- **Type Safety**: Full TypeScript support

## Quick Start

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Set up environment variables**:
   Create a `.env.local` file:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:3001
   ```

3. **Start development server**:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:3000`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Technologies

- **Next.js** - React framework with App Router
- **TypeScript** - Type safety
- **React** - UI library
- **CSS Modules** - Scoped styling
- **Context API** - State management
